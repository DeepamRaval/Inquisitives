Safe-Scan Lite
Safe-Scan Lite is a lightweight security application designed to protect users from QR code phishing attacks (Quishing). It scans QR codes and analyzes the embedded links to determine whether they are safe or potentially malicious before users open them.

Features
QR Code Scanning – Quickly scans QR codes using your device camera
Link Analysis – Extracts URLs from QR codes and checks them for suspicious patterns
Quishing Detection – Identifies phishing or fake links commonly used in QR-based scams
User Safety Alerts – Clearly notifies users whether a link is safe or risky
Lightweight & Fast – Minimal resource usage with quick scan results

How It Works
1. The user scans a QR code using Safe-Scan Lite.
2. The app extracts the URL embedded in the QR code.
3. The link is analyzed using security checks (domain validation, suspicious keywords, URL structure, etc.).
4. The app displays a Safe or Warning result before the user proceeds.

Use Cases
Scanning QR codes on posters, menus, payment slips, or public places
Preventing access to malicious or fake websites
Enhancing digital awareness and online safety

Technology Stack
Frontend: HTML, Dart, CMake, Swift
Backend: C, C++
Others: Vercel, Median, XCode, Android Studio

Disclaimer
Safe-Scan Lite provides risk assessment, not a 100% guarantee. Users are advised to stay cautious while opening unknown links.
